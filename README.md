# netflix-clone
A simple and responsive Netflix landing page clone built using HTML and CSS. This project replicates the UI of the Netflix homepage to practice frontend development skills.
_______________
Clean and modern Netflix-inspired UI
Responsive design (works on mobile, tablet, desktop)
Hero section with background image
Sign-in button UI
Sections showcasing features (TV, mobile, kids, etc.)
Email input field for signup simulation
Smooth layout using pure CSS
